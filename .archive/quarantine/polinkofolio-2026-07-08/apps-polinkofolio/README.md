# Polinkofolio App Surface

Polinkofolio is the Polinko-native public notebook surface.

This directory is reserved for the presentation layer that can later render
curated Polinko field notes, diagrams, and source-led snippets. The source
truth stays in Polinko research, eval, transcript, and local-evidence surfaces.

## Current Shape

- `docs/polinkofolio/` holds the planning and source map.
- `docs/public/` holds current public Polinko reading surfaces.
- `docs/peanut/` holds private research planning and transcript routing.
- `.local/` holds local evidence, eval packets, and runtime databases.
