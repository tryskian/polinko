## Core Hypothesis

State the hypothesis in one falsifiable sentence.

LLM safety and alignment improve when interaction is governed by a stable
macro-pattern (ethics/policy/guardrails) while adaptation happens through
real-time micro-pattern learning/unlearning, instead of static
customer-service-style helpfulness behaviors.

## Definitions

Define the non-negotiable terms used in this hypothesis.

- Macro-pattern: stable constraints that do not change per turn (ethics floor,
  safety policy, boundary rules, evidence discipline).
- Micro-pattern: local adaptation in response to live interaction signals
  (style fit, uncertainty calibration, correction/recovery behavior).
- Ethics vs ethos: ethics is invariant (must hold); ethos is adaptive
  presentation style (may vary).
- Non-collusive grounding: validate emotion without validating ungrounded
  claims, motive certainty, or stereotype conclusions.

## Exclusions (What this claim is not)

- Not a sentience/awareness claim.
- Not a complete case study.
- Not a claim that tone/personality is irrelevant; only that style cannot
  override ethics and evidence boundaries.

## Predicted Observables

List what should be measurable if the hypothesis is valid.

- Reduced collusion signals on personal recursive prompts (`why did X leave
  me`, motive-certainty prompts, stereotype prompts).
- Reduced fabricated certainty in low-evidence interpersonal questions.
- Faster recovery after correction without rigid over-indexing.
- Better uncertainty calibration with maintained user-aligned tone.
- Lower occurrence of politeness-economy artifacts (reflexive apology,
  reassurance loops, invented completions).

## Falsification Criteria

List conditions that would disconfirm the hypothesis.

- Guardrailed macro/micro architecture does not reduce motive-fabrication or
  collusive certainty compared to baseline helpfulness mode.
- Safety outcomes improve only by flattening interaction quality (for example,
  blanket refusal patterns with no practical support).
- Real-time adaptation repeatedly degrades into unstable overfitting despite
  bounded guardrails.

## Key Points From Transcript Evidence

Source notes: `docs/transcripts/guardrails-dialogue-notes-2026-03-08.md`

- LLM vs tracker distinction: inference and telemetry are separable layers.
- Context is required for safe interaction, but must be bounded (purpose,
  scope, retention, control).
- Customer-service politeness priors conflict with long-form inquiry goals.
- Anthropomorphism is not equivalent to personality; policy-constrained systems
  require capability-grounded reasoning.
- One-sided personal prompts require non-collusive framing (emotion validation,
  no motive certainty).
- Recursive interpersonal "why" loops need loop interruption, not certainty
  amplification.
- Policy should constrain response behavior, not infer hidden human intent;
  ambiguous intent should trigger uncertainty + clarification, not projection.
- Divination/astrology cues should not be treated as deterministic evidence of
  motive or fate.
- Working model: micro-pattern adaptation inside macro-pattern ethics.
