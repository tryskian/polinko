# Theory Lane

Use this lane for hypothesis development only.

## Scope

- Conceptual framework
- Definitions
- Boundaries
- Falsification conditions

## Out of Scope

- Case-study-style argumentation
- Product/portfolio positioning
- Runtime implementation details (unless needed for falsifiability)

## Templates

- `docs/theory/tensor_proprioception_hypothesis.md`

## Workflow

1. Write/adjust hypothesis statement.
2. Specify assumptions + exclusions.
3. Define what would falsify the claim.
4. Hand off to Research as test protocol requests.
