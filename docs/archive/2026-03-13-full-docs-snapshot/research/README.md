# Research Lane

Use this lane to test theory claims with reproducible evidence.

## Scope

- Experiment design
- Protocol execution
- Eval metrics and artifacts
- Failure analysis and remediation

## Inputs

- Theory hypothesis IDs (for example `H-001`)

## Outputs

- Experiment records
- Evidence artifacts in `docs/portfolio/raw_evidence/`
- Eval reports in `eval_reports/`

## Templates

- `docs/research/experiment_template.md`

## Minimum record quality

Each experiment should capture:
- `experiment_id`
- linked `hypothesis_id`
- protocol
- command(s)
- artifact path(s)
- outcome and next action
