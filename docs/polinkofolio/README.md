# Polinkofolio

Polinkofolio is the Polinko-native public notebook surface.

It gathers polished fragments from Polinko discourse, evidence notes, diagrams,
and research maps into a public reading path while keeping the repo as the
source of truth.

## Source Lanes

| Lane | Current source |
| --- | --- |
| Field notes | `docs/public/` |
| Research notes | `docs/research/` |
| Diagrams | `docs/public/diagrams/` |
| Transcript notes | `docs/peanut/transcripts/` |
| Private planning | `docs/peanut/` |
| Eval packets | `.local/eval_cases/` |
| Eval reports | `.local/eval_reports/` |
| Manual eval warehouse | `.local/runtime_dbs/active/manual_evals.db` |

## Surface Contract

- Start from source evidence.
- Promote curated fragments.
- Keep public language source-led and lightweight.
- Keep repo mechanics aligned with the shared operator template.
